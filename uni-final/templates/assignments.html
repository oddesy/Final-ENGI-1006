Assignments

  • HW 1A - We wrote pseudocode for an algorithm to convert binary to decimal,; converted binary to signed binary, one's complement, two's complement, and octal; added positive binary numbers; wrote an algorithm to calculate interest; and completed a truth table with logical operators.
  • HW 1B - We wrote a program to take user input and perform operations on the input. We also wrote various other algorithms to call on lists.
  • HW 2A - We had to understand a sorting algorithm; understand the time complexity of a binary sort algorithm for a phone book; and iterate through a list and understand the time complexity of the algorithm.
  • HW 2B - We had to simulate a falling object in Earth's gravity.
  • HW 3A - We had to write pseudocode and provide the time complexity for certain algorithms and implement functions recursively.
  • HW 3B - We had to build a module that would percolate down through a grid and write some of the beginning functions for the module that would analyze data from a dataset.
  • HW 4 - We had to model a class with a Teacher, Students, and Assignments. The program had to somewhat randomly assign students grades for each assignment that the Teacher (the user) would assign). We also had to plot all of the class information before program termination.
  • HW 5 - We had to finish our module that would analyze a dataset. It needed to provide complex statistics on each column of the dataset, plot multiple graphs of the data, plot a heatmap of the data, and project whether a row was malignant or benign.
  • Midterm - For the midterm, we had to implement three algorithms to preform on .ppm files (images). The first algorithm had to take three images and remove the photoshopped portions and return the unedited image. The second algorithm needed to turn the images into black and white images. The third algorithm needed to flip the image horizontally.
